# StudySathi - Your Study Companion
## Overview
StudySathi is a **web-based study companion** designed to help students stay organized, focused, and productive. It combines **task management, focus timers, and study analytics** in a single intuitive platform.


## Inspiration
Managing study schedules, tasks, and focus can be overwhelming. We wanted a tool that motivates students, helps them plan effectively, and tracks their progress â€” all in one place.


## What it does
- **Task Management:** Add, prioritize, filter, and mark tasks as completed.  
- **Pomodoro Timer:** Stay focused with 25-minute study sessions and breaks.  
- **Study Suggestions:** Get recommendations on what to study next.  
- **Analytics:** Track daily and weekly study hours, total tasks, and completed tasks.  
- **Dark/Light Mode:** Toggle between themes for better usability.


## How we built it
- **Frontend:** HTML, CSS, JavaScript  
- **Data Storage:** Browser Local Storage for tasks and study tracking  
- **Responsive Design:** Works on mobile, tablet, and desktop  
- **Interactive Features:** Dynamic task rendering, timers, notifications, and theme toggle  


## Challenges we ran into
- Dynamic task prioritization based on deadlines  
- Implementing a responsive and interactive Pomodoro timer  
- Smooth dark/light theme switching without breaking UI  
- Persistent data storage with Local Storage  


## Accomplishments
- Fully functional **study companion** with task, timer, and analytics features  
- Intuitive, **user-friendly interface**  
- Dynamic **study suggestion algorithm**  
- Responsive and accessible across devices  

## What we learned
- Building a complex **frontend application** with multiple interactive components  
- Using **Local Storage** for persistent state management  
- Handling **UI/UX challenges** such as responsive layouts and theme toggling  
- Implementing **priority and feedback mechanisms** in productivity apps
